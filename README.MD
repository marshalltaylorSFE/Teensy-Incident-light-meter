Teensy Incident Light Meter
--------

<img src="https://raw.githubusercontent.com/marshalltaylorSFE/Teensy-Incident-light-meter/master/MainObject.jpg" alt="Product" width="40%" height="40%">

### Software

Requires the following libraries:

* [Teensyduino](https://www.pjrc.com/teensy/teensyduino.html)
  * Uses included encoder library
* [uCModules library](https://github.com/marshalltaylorSFE/uCModules)
* [TeensyView library](https://github.com/sparkfun/SparkFun_TeensyView_Arduino_Library)
* [TSL2561 library](https://github.com/sparkfun/SparkFun_TSL2561_Arduino_Library)

### Hardware

* [Teensy 3.2](https://www.sparkfun.com/products/13736)

### Mechanical

Uses the following:

* [112x61x31mm Enclosure](https://www.sparkfun.com/products/13839)
* [Snappable Breadboard (x2)](https://www.sparkfun.com/products/13268)
* A ping pong ball
* A 3D printed bezel for the ball and light sensor (see /Bezel3DModel)

The 3D model is a bezel for mounting a ping pong ball to a panel.  The model also has screw holes for the light sensor.  Units are in inches, use 2.54 as a ratio for scaling

Case is cut ad-hoc.
